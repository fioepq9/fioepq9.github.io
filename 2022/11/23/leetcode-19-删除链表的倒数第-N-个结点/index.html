原题

提示：

  • 链表中结点的数目为 sz
  • 1 <= sz <= 30
  • 0 <= Node.val <= 100

解题思路

该题的关键在于获取到要删除结点的前一个结点。

考虑到要删除的结点可能为头结点，建议定义一个哑结点作虚拟头结点来简化处理。

两个结点都从头结点出发，一个结点先走 n 步，然后两个结点同时前进，当第一个结点走到末尾时，第二个结点即为要删除结点。
